Projects

  • Pulley System Cable Car With Mechanical Braking – This project is a fully mechanical cable car system that moves along a string by the use of a unidirectional, compound pulley system. The mechanical braking system is triggered by a station peg which causes a trip lever to set the wedge in between the cable and a fixed brake pad. Once the wedge is fully set in place, it triggers a falling weight to slowly pull the wedge out until the car continues moving again.

    How it Works

    The car glides along a cable, powered by a pulley and counterweight system which is attached to the car body. When the car reaches a station peg, it causes the trigger to spin counter clockwise which spins the popsicle stick that is connected to the wedge clockwise. When the popsicle stick connected to the wedge rotates, it hits the cable forcing the wedge backwards between the brake pad and cable, stopping the car. When the wedge is fully set in place, the string attached to the wedge tightens removing the pin and dropping the weight. The weight will slowly pull out the wedge until the car resumes movement.
