Projects

  • Pulley System Cable Car With Mechanical Braking – This project is a fully mechanical cable car system that moves along a string by the use of a unidirectional, compound pulley system. The mechanical braking system is triggered by a station peg which causes a trip lever to set the wedge in between the cable and a fixed brake pad. Once the wedge is fully set in place, it triggers a falling weight to slowly pull the wedge out until the car continues moving again.
